Desarrollo de una SPA (Single Page Application) utilizando React y Redux para el Front End. Todos los componentes fueron desarrollados con CSS puro sin uso de librer√≠as externas.
La SPA consume datos de una API (https://pokeapi.co) a traves de un Back End desarrollado en NodeJS, utilizando Express y PostgreSQL como base de datos.
